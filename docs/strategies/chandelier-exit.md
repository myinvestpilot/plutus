# 吊灯止损策略

吊灯止损就是一个会自己调整的止损方法。当股票波动大的时候，止损线离得远一点；波动小的时候，止损线离得近一点。这样既不会被正常波动震出去，又能保护你的利润。

## 怎么工作的

### 基本思路
这个策略用ATR（平均真实波幅）来判断股票的波动情况：
- 股票最近波动大，止损线就设得远一点
- 股票最近波动小，止损线就设得近一点
- 当股票涨的时候，止损线也跟着往上抬
- 这样既能保护利润，又不会轻易被震出去

### 计算方法
很简单，就是用最近一段时间的最高价，减去ATR乘以一个倍数：

**止损价 = 最高价 - ATR × 3**

- 最高价：比如最近22天的最高价
- ATR：最近22天的平均波动幅度
- 倍数3：可以调整，3是比较常用的

比如某股票最高价是100元，ATR是2元，那止损价就是100-2×3=94元。

## 为什么有用

### 1. 自动适应波动
- 股票波动大的时候，不会因为正常波动就卖出
- 股票波动小的时候，止损线跟得紧一些
- 不用你去猜今天该设多少止损

### 2. 保护利润
- 股票涨的时候，止损线也跟着往上抬
- 就算后来跌了，也能保住一部分利润
- 不会出现"纸面富贵一场空"的情况

### 3. 减少情绪干扰
- 完全按公式计算，不用你主观判断
- 该卖就卖，不会因为舍不得而拖延
- 避免"再等等看"的心理

## 参数怎么设置

### ATR天数
- **14天**：适合短线，反应比较快
- **22天**：最常用，比较平衡
- **50天**：适合长线，比较稳定

### 倍数选择
- **2倍**：止损比较紧，但容易被震出去
- **3倍**：最常用，比较合适
- **4倍**：止损比较松，但亏损可能较大

**怎么选择**：
- 如果你容易被假跌震出去，用4倍
- 如果你怕亏太多，用2倍
- 大部分情况下，3倍就够了

### 3. 周期选择
**时间框架**：
- 日线：适用于中长期交易
- 4小时线：适用于短期交易
- 周线：适用于长期投资

## 怎么用

### 什么时候买入
这个策略主要是用来止损的，不是用来决定买入的。你可以：
- 根据其他信号买入股票
- 买入后立即设置吊灯止损
- 比如买入价100元，ATR是2，那么止损就设在94元

### 怎么调整止损
**重要规则**：止损线只能往上抬，不能往下降
- 每天收盘后重新计算
- 如果新的止损价比昨天高，就调整上去
- 如果新的止损价比昨天低，就保持昨天的
- 一旦价格跌破止损线，立即卖出

### 什么时候卖出
**主要出场信号**：
- 股价跌破止损线就卖出
- 不要犹豫，不要"再看看"
- 严格执行，保护本金最重要

**其他情况**：
- 如果基本面出现重大变化，也可以考虑卖出
- 但主要还是按止损线执行

## 什么时候适合用

### 趋势比较明确的股票
- 正在上涨的股票，用来保护利润
- 趋势性比较强的股票效果更好
- 那种天天上下乱跳的股票不太适合

### 中长期持有
- 适合准备持有几个月的投资
- 短线交易用这个可能太慢
- 长期投资用来控制风险很不错

### 组合管理
- 可以给组合里的每只股票都设置
- 控制单只股票的最大亏损
- 避免某只股票拖累整个组合

## 一些改进方法

### 按百分比调整
不同价位的股票，可以按百分比来设置：
- 100元的股票，3%的止损就是97元
- 10元的股票，3%的止损就是9.7元
- 这样对不同价位的股票比较公平

### 结合其他指标
- 可以配合均线使用：股价在均线上方才买入
- 可以配合RSI：超买的时候止损设紧一点
- 但不要搞得太复杂，简单实用就好

## 使用技巧

### 选股票要注意
**适合的股票**：
- 有明确趋势的股票
- 波动不要太疯狂的
- 成交量够大，好买好卖的

**不适合的股票**：
- 天天横盘震荡的
- 波动特别大的妖股
- 成交量很小的冷门股

### 心理准备
- 会有一些小的止损，这很正常
- 重要的是保护本金，避免大亏
- 严格按规则执行，不要心软

## 风险和注意事项

### 主要风险
- **震荡市容易被反复止损**：横盘的时候用这个策略比较难受
- **假突破会造成损失**：有时候突破是假的，买入后就开始跌
- **参数设置不合适**：倍数太小容易被震出去，太大亏损会比较多

### 怎么控制风险
- 不要把所有钱都投一只股票
- 单只股票的止损最好不超过总资金的2%
- 定期检查策略是否还有效
- 震荡市的时候可以考虑暂停使用

## 实际效果怎么样

### 历史表现
根据回测数据：
- 能够有效控制大的亏损
- 在趋势市场中表现较好
- 最大回撤通常能控制在合理范围内
- 震荡市中会有一些小的止损

### 不同市场环境
- **牛市**：能跟上大部分涨幅，保护利润
- **熊市**：能够及时止损，避免大亏
- **震荡市**：会有一些来回的小亏损

## 总结

吊灯止损策略就是一个会自动调整的止损方法：

**主要好处**：
- 不用你天天想该设多少止损
- 能根据股票的波动自动调整
- 帮你保护利润，控制风险
- 减少情绪化的决策

**适合的人**：
- 想要系统化止损的投资者
- 容易情绪化交易的人
- 中长期投资者
- 重视风险控制的人

**使用要点**：
- 参数不要设得太复杂
- 严格按规则执行
- 接受小的止损是正常的
- 重要的是保护本金
