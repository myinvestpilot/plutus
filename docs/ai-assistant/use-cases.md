```
📚 夏普比率(Sharpe Ratio) 详解

**定义**：
夏普比率是衡量投资组合**每单位风险所获得超额收益**的指标，由诺贝尔经济学奖得主威廉·夏普提出。

**计算公式**：
```
夏普比率 = (组合收益率 - 无风险收益率) / 组合波动率
```

**指标解读**：
• **>1.0**：优秀，风险调整后收益很好 ✅
• **0.5-1.0**：良好，风险收益比较合理 🟢
• **0-0.5**：一般，收益勉强覆盖风险 🟡
• **<0**：较差，承担风险却获得负收益 ❌

📊 **实际应用示例**：

**组合A**：
• 年化收益率：15%
• 无风险收益率：3%  
• 波动率：20%
• 夏普比率 = (15%-3%)/20% = 0.60

**组合B**：
• 年化收益率：12%
• 无风险收益率：3%
• 波动率：15%  
• 夏普比率 = (12%-3%)/15% = 0.60

**结论**：两个组合夏普比率相同，风险调整后收益相当

🎯 **评估投资组合应用**：

**1. 组合比较**：
• 比较不同策略的风险调整收益
• 夏普比率高的组合更优秀
• 注意：单看收益率可能误导

**2. 风险预算**：
• 高夏普比率组合可以承担更多风险
• 用于确定最优风险敞口水平

**3. 基准比较**：
• 与市场指数(如沪深300)夏普比率对比
• 评估主动管理的价值

**在策引平台的应用**：
• 策略回测报告中会显示夏普比率
• 用于评估策略的风险调整收益
• 帮助选择最优参数组合

💡 **使用技巧**：
• 结合最大回撤一起看，全面评估风险
• 关注长期夏普比率，短期数据可能失真
• 不同市场环境下的夏普比率可比性有限

**相关指标**：
• 索提诺比率：只考虑下行波动率
• 卡玛比率：用最大回撤替代波动率
• 信息比率：相对基准的超额收益风险比
```
